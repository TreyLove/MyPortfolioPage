The Unanswered Questions

The Unanswered questions is a website that offers individuals who are new to philosophy the tools to expand their knowledge. Some of the tools we currently offer to our users are a quote quiz, qoute selector as well as philosopher profiles that give a short breakdown on each curated philosopher along with videos and a wikipedia link t learn more. The tech used in this website were HTML, CSS, JavaScript, jQuery and Tailwind. The API's used were the wikipedia API, the YouTube API and a quote API.

Simple Weather

Simple weather is a simple website that offers the user a quick and easy format to get weather for any location in the world. The technology used was HTML, CSS, JavaScript and Moment.js. The API's that were used are openCageData and openWeatherMaps

Daily Planner

This Daily planner offers a simple scheduling solution for the 9-5 workforce. It allows users to make logs for each hour of the day to track their tasks and also tracks the current day and hour with moment.js. The technology used in this website is HTML, JavaScript, BootStrap, Moment.js and CSS.
